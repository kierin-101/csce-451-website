Projects I've Worked On

Pitivi

Pitivi is a free and open-source video editor for Linux. It is written in Python and uses the GStreamer multimedia framework. My team "contributed" (the changes were never submitted in a merge request) a new feautre that allowed the user to essentially bookmark points in their video timeline, give the bookmakrs names, do things like that.

(It was kinda boring, but it was a good learning experience. I learned a lot about the Git workflow and how to work with a team of developers. I learned a lot about the GStreamer framework and how to work with it. I also also learned that I don't like working with GStreamer.)
